## 6. 当前第一版结论

### 6.1 为什么它能走成
- 不是单一“日级净流入持续正”这么简单；
- 更像是：
  1. 启动前 L2 已有资金埋伏；
  2. 主升中超大单承接显著；
  3. 高位阶段盘口 `oib` 仍维持正向支撑；
  4. 允许高换手，但没有直接塌。

### 6.2 为什么它高位还能稳
- 第一版理解：
  - **高位能稳住**，关键不是“每天都大买爆买”，
  - 而是“卖压来时，盘口仍有人接，且买侧挂单结构没有明显崩坏”。
- 这类票后续更值得重点盯：
  - `超大单净额`
  - `oib`
  - `撤买 / 撤卖`
  - 高位大阴日后的修复能力
