## 2. 为什么需要本卡
- `2026-03-22 ~ 2026-03-24` 期间，散户模块经历了多轮方案调整：先做 `freshness / 指标 / 联动`，后又切到 `热度主导 + 统一事件流`，再进一步收敛为 `股吧单源 + 星标日更 + 日级 AI 解读`。
- 过程中保留了多张 STG / REQ 卡；这些卡用于记录决策过程仍有价值，但如果未来只想回答“当前线上到底是什么”，继续逐张翻卡会很费劲。
- 本卡的职责是：
  1. 用**当前真实落地**覆盖“过程中的拟方案”；
  2. 明确哪些内容已经完成、哪些被调整、哪些暂未做；
  3. 为后续发版、回滚、继续扩展提供单一母卡。

## 3. 当前真实落地（以代码为准）

### 3.1 模块定位
- 首页模块正式名称：`散户一致性观察`
- 业务定位：**股吧讨论热度 + 相对热度 + 价格联动 + 星标股 AI 日评**
- 不承担盘中高频交易信号职责。

### 3.2 当前正式数据源
- **只保留东方财富股吧**
- `sentiment_events` 是正式事件底座；
- 旧 `sentiment_comments` 仍保留，用于兼容与懒回填；
- 当前首页正式可见数据，仍以：
  - `股吧主帖正文/标题`
  - `旧表兼容回填`
  为主。

### 3.3 当前正式接口
- `GET /api/sentiment/overview/{symbol}?window=5d|20d|60d`
- `GET /api/sentiment/heat_trend/{symbol}?window=5d|20d|60d`
- `GET /api/sentiment/feed/{symbol}?window=5d|20d|60d&sort=latest|hot&limit=50`
- `GET /api/sentiment/daily_scores/{symbol}?window=20d|60d`
- `POST /api/sentiment/crawl/{symbol}`
- `POST /api/internal/sentiment/backfill_starred/{symbol}`
- `POST /api/internal/sentiment/score_daily/{symbol}?trade_date=YYYY-MM-DD`

### 3.4 当前页面结构
- 左：顶部指标卡 + 上下双层热度主图
  - 上图：价格 + AI 情绪分
  - 下图：热度柱 + 相对热度 + AI 一致性 + AI 温度
- 中：AI 日评摘要 + 观察提示 + AI 日评日志
- 右：最近 50 条股吧原文，支持 `最新 / 最热`

### 3.5 当前窗口与默认行为
- 窗口：`5D / 20D / 60D`
- 默认窗口：`20D`
- 当前查看股票：
  - 打开页面自动抓一次
  - 停留 30 分钟再补抓一次
- 右侧原文流与主图窗口保持一致。

### 3.6 当前 AI 日评口径
- 只对**星标股**做日级 LLM 解读缓存；
- 输出字段：
  - `sentiment_score (-100~100)`
  - `direction_label`
  - `consensus_strength`
  - `emotion_temperature`
  - `risk_tag`
  - `summary_text`
- 候选样本不是全量送模，而是先筛高价值主帖样本后送模；
- 首页读接口只读缓存，不实时调模。

### 3.7 当前调度与补数
- 新加星标时：
  - 立即触发股吧抓取
  - 立即触发历史回补：先 `20D`，成功后继续尝试 `60D`
- 定时任务（星标股）：
  - `08:40` 盘前抓取
  - `15:30` 盘后补抓
  - `21:00` 夜间终补
  - `15:40` 盘后日评分
  - `21:10` 夜间终版日评分

## 4. 与原规划相比，哪些被调整了

### 4.1 已被明确收敛的内容
- **雪球 / 同花顺**
  - 原规划曾预留/尝试两源；
  - 当前版本已明确收敛为**股吧单源**，不再作为本轮正式范围。
- **AI 仅占位**
  - 原 `V2` 规划里 AI 区一度只保留占位；
  - 实际已升级为：星标股支持**日级 AI 解读 + 日志 + 手动立即总结**。
- **右侧来源 Tab**
  - 原规划里曾预留 `全部 / 股吧 / 雪球 / 同花顺`；
  - 当前页面实际为**固定股吧原文流**，不再展示来源切换。

### 4.2 已按实际实现替代原表达的内容
- 主图不再显示叙事类文字标签；
- 风险标签留在：
  - 顶部卡片
  - Tooltip
  - AI 日评日志
- 图内只保留数值因子，不再把“叙事发酵/低热观望”强行贴到图上。
