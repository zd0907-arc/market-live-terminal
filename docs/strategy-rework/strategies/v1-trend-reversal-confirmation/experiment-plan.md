# v1 实验计划

## 实验窗口

首轮窗口：`2026-03-02 ~ 2026-04-24`

原因：该窗口同时具备 L2 成交资金和 L2 挂单数据。

## 对照基线

v0 二次确认策略：

- 胜率：32.54%
- 平均收益：1.54%
- 中位收益：-7.40%

v0 累计超大单保护出场：

- 胜率：37.01%
- 平均收益：3.26%
- 中位收益：-6.75%

## v1 首轮目标

不要求一次达到实盘标准，先看是否改善：

- 胜率 > 37%
- 平均收益 > 3.26%
- 中位收益高于 -6.75%
- 交易数量不能少到失去意义

## 实验输出

每次实验必须输出：

1. 每日候选清单。
2. 每只票发现日、启动确认日、回调确认日。
3. 买入日、卖出信号日、卖出日。
4. 每笔收益、最大浮盈、最大回撤、持有天数。
5. 汇总胜率、平均收益、中位收益。
6. 与 v0 对照。

## 首轮策略流程

```text
T日发现：
  pre20/pre5 价格偏弱或未明显上涨
  资金价格背离为正
  不要求当天大涨

T后观察：
  最多观察 5 日启动质量
  启动3日资金和价格质量合格才进入等待回调

回调确认：
  回调不破坏结构
  回调期间 L2 主力/超大单承接仍在
  挂单支撑不弱

买入：
  回调确认日下一个交易日开盘买入

卖出：
  累计 L2 超大单转弱 + 资金/挂单风险同步出现
  或硬止损/最长持有兜底
```

## 首轮复盘后的 v1.1 方向

v1.1 优先优化：

1. 持有期累计超大单/主力资金未转弱时，不轻易卖。
2. 持有期 L2 正流入切片、主动买强度、挂单承接转弱时，才确认风险。
3. 发现日至买入日前过热降权，避免“确认后仍然追高”。
4. 保留同票去重/冷却机制作为更接近实盘的评估口径。
