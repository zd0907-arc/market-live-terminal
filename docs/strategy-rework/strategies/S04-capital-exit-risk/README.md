# S04-capital-exit-risk：资金撤退与风险识别

## 目标

沉淀可复用的退出/风险模块：

```text
累计超大单实际下降
暴烈超大单流出
撤梯子诱多
压单出货
挂单与成交背离
抢跑式风险
```

## 当前状态

当前最有价值的方向是组合风险，不是单因子硬过滤。

候选观察模块：

```text
S04-M01-observe-combined-risk-stack
```

核心规则：

```text
R1~R5 中同时出现 >=2 个风险信号
```

其中：

- R1：启动期撤买/新增买接近异常防线。
- R2：启动期 OIB/CVD 背离。
- R3：确认日出货分偏高。
- R4：确认日超大单和主力同时为负。
- R5：弱启动 + 回调承接差。

## 当前验证结论

- 在 S01-M04B 成熟交易上，组合风险过滤后胜率、平均收益、中位收益都提升。
- 扩展到 v1.3/v1.4 多个样本后，方向仍然有效。
- 但存在误杀大赢家，且样本期仍只有 2026-03~04 的 L2 挂单数据。

所以当前只作为：

```text
观察型风险模块 / 复盘解释字段
```

暂不作为：

```text
S01 默认硬过滤
```
