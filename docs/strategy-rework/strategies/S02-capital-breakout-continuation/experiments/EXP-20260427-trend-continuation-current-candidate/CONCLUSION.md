## 买点规则

在观察池和严格高质量回踩基础上，新增真承接过滤：

```text
确认日主动买入强度 > 0
确认日主力净流入比例 >= 0
```

目的是过滤 `sz002633` 这种“形态像回踩，但资金不承接”的弱反抽。

## 卖点规则

新增趋势中继专用退出：

```text
已有累计超大单峰值
累计超大单从峰值回撤 >= 25%
当日超大单净流出 / 当日成交额 <= -5%
当前收盘仍不亏钱
=> 当日收盘发信号，次一交易日开盘卖出
```

目的是处理 `sz000767` 这种涨停日超大单大额派发的问题。
