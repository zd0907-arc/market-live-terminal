# 严格高质量回踩：卖点优化分析

## 结论

旧卖点对“单日大额超大单派发”反应偏慢。

建议新增趋势中继专用卖点：

```text
如果持仓已有累计超大单峰值，且：
1. 累计超大单从峰值回撤 >= 25%
2. 当日超大单净流出 / 当日成交额 <= -5%
3. 当前收盘仍不亏钱
则当日收盘发出卖出信号，次一交易日开盘卖出。
```

这个规则的关键变化：

```text
旧规则看“连续3天累计下降”；
新规则看到“单日大额派发”就先跑。
```

## 关键案例

`sz000767` 是典型：

```text
2026-03-27 股价涨停，但超大单净流出约 -1.49 亿，占当日成交额 -6.24%。
同时累计超大单从峰值回撤 30.6%。
旧规则等到连续3日下降，实际 2026-04-01 才卖。
新规则 2026-03-27 收盘识别，2026-03-30 开盘卖，收益从 8.01% 提高到 28.11%。
```

## 注意

`sz000967` 说明阈值不能太敏感。
如果把 -5% 放宽到 -4.5%，会在 4/15 提前卖，当前样本收益更高，但有过拟合风险。
建议先用 -5% 作为保守版本。
