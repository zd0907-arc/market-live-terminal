# 严格高质量回踩：赚钱/亏损归因

## 结论

从“只追求高胜率、高收益”的目标看，当前最有价值的不是扩容，而是进一步过滤出：

```text
观察池质量靠前
启动日修复质量强
前20日涨幅不过度透支
```

在当前两个月样本里，一个候选过滤口径是：

```text
观察池排名 <= 15
启动/修复质量分 >= 45
前20日涨幅 <= 30%
```

结果：

| 交易数 | 胜率 | 平均收益 | 中位收益 | 最大亏损 |
|---:|---:|---:|---:|---:|
| 8 | 87.50% | 13.78% | 10.46% | -8.32% |

注意：样本只有 8 笔，只能作为下一步假设，不能直接投产。

## 为什么这么过滤

Top20 里亏损票的中位特征：

```text
观察池排名更靠后：15.5
启动日修复质量弱：37.47
前20日涨幅更高：23.29%
买入后几乎不给浮盈：最大浮盈中位数 0.95%
```

大赚票的中位特征：

```text
观察池排名更靠前：6.5
启动日修复质量更强：58.34
前20日涨幅略低：20.12%
买入后很快给浮盈：最大浮盈中位数 28.83%
```

业务解释：

```text
亏损票不是确认日指标明显差，而是入池质量和启动修复质量不够。
它们看起来也回踩了，但更像弱趋势里的反抽，不像强趋势里的健康回踩。
```

## 当前候选规则

```text
强趋势观察池
+ 严格高质量回踩确认
+ 观察池排名 <= 15
+ 启动/修复质量分 >= 45
+ 前20日涨幅 <= 30%
```
