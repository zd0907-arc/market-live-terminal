# 趋势中继策略：下一版买点方案

## 结论

趋势中继不能“入池后次日直接买”。

下一版拆成两层：

```text
强趋势观察池
+ 二次买点确认
```

## 为什么要改

第一版趋势中继原型能抓到很多强势股，但交易结果差：

| 口径 | 交易数 | 胜率 | 平均收益 | 中位收益 |
|---|---:|---:|---:|---:|
| 成熟交易 | 263 | 39.92% | 0.38% | -6.56% |

失败主要不是卖早，而是买点太粗：

```text
很多票入场后 3~5 天直接走弱，最大浮盈很小，然后触发硬止损。
```

亏损票买入后前三天的典型特征：

| 指标 | 大赢家中位数 | 大亏票中位数 |
|---|---:|---:|
| 买入后3日累计超大单 | +4255万 | -5919万 |
| 买入后3日累计主力 | +2160万 | -9887万 |
| 买入后3日最低回撤 | -2.13% | -8.74% |

## 观察池定义方向

观察池只回答：

```text
这只票是否已经处在强趋势中，并且资金还没明显撤退？
```

不直接回答“明天能不能买”。

观察池候选特征：

```text
近一段时间已有明显涨幅
价格仍接近阶段高位
累计超大单未明显从峰值撤退
主力/超大单资金没有连续恶化
没有明显放量砸盘
```

## 二次买点候选

### 1. 回踩均线承接

```text
回踩 5 日线或 10 日线附近不破
当日或次日主力/超大单转正
下跌缩量，修复放量
```

业务含义：趋势还在，回调是洗盘或换手，不是出货。

### 2. 缩量震荡后放量阳线

```text
连续 2~5 日缩量整理
累计超大单没有明显撤退
随后放量阳线重新上攻
```

业务含义：中继整理结束，资金重新发动。

### 3. 强分歧后修复

```text
出现大阴线或剧烈分歧
但累计超大单没有被砸成明显负撤退
1~3 日内快速修复关键跌幅
```

业务含义：分歧不是抢跑，资金还在场内。

### 4. 前高附近震荡不出货

```text
靠近前高或平台上沿
成交活跃但不放量滞涨
新增卖压、撤买单、出货分不共振恶化
```

业务含义：不是上方派发，而是在蓄势突破。

## 验证指标

下一版不能只看胜率，还要同时看：

```text
交易数
胜率
平均收益
中位收益
最大亏损
硬止损比例
强势 Top50 覆盖
涨幅>=30% 覆盖
涨幅>=50% 覆盖
强势股中误亏比例
非强势股误抓比例
```

## 采纳标准

趋势中继下一版至少要满足：

```text
覆盖明显高于资金流回调稳健策略；
中位收益不能长期为负；
硬止损比例必须明显下降；
不能靠极少数大牛拉高平均收益。
```


## 当前聚焦方向

先聚焦：

```text
严格高质量回踩确认
```

当前规则含义：

```text
趋势已经存在；
出现回踩均线承接；
确认日不过度放量；
确认日撤买单低；
确认日涨幅不过热。
```

已验证：Top10 质量较好，Top20 覆盖更高但质量下降。

## 暂存待研究方向

以下方向先记录，不立刻展开：

```text
强势主升/平台突破中继
前高附近震荡不出货
强分歧后修复买点
确认后 1~2 日资金留场二次确认
Top20 新增亏损票归因
消息事件重估策略
```
