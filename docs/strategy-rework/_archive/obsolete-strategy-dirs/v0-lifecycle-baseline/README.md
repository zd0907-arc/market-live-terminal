# v0 生命周期基线策略

## 定位

这是旧版实验基线，不再作为主研究方向，但保留用于对照。

核心逻辑：

- 候选：吸筹/攻击/修复/出货等生命周期分。
- 交易方式：信号次日买、或二次确认买。
- 出场：止损、出货分、最长持有。

## 关键实验结果

### 信号次日进场

来源：`../../experiments/20260426-quick-validation/direct_trades.csv`

## v0 暴露的问题

1. 候选仍偏追急涨。
2. 中位收益明显为负，靠少数大牛拉均值。
3. 没有充分利用启动前资金价格背离。
4. 挂单行为没有阶段化建模。
