# v1 趋势反推确认策略

## 一句话定位

从“大趋势正样本”反推：先找启动前价格弱但资金/价格背离为正的票，再等启动质量和回调承接确认，持有期间用累计 L2 超大单判断资金是否撤退。

## 为什么做 v1

v0 的问题是容易追急涨。v1 改为：

```text
启动前压住/洗盘/横盘
+ 资金价格背离为正
+ 启动 3 日质量确认
+ 回调承接确认
+ 累计超大单未撤不轻易卖
```

## 当前不做什么

- 不接新闻/题材解释。
- 不做仓位管理。
- 不做分批止盈。
- 不改前端页面。

## 交易策略形态

### 发现日 T

只使用 T 及 T 以前数据，计算：

1. 启动前 20 日价格表现。
2. 启动前 5 日价格表现。
3. 启动前资金价格背离。
4. 当日是否有初始异动，但不要求暴涨。

### 观察期

T 后最多观察若干交易日，等待：

1. 启动 3 日质量成立。
2. 首次回调不破坏趋势。
3. 回调时资金/挂单承接仍在。

### 买入

确认信号日后的下一个交易日开盘买入。

### 卖出

持有期间以累计 L2 超大单为核心：

```text
累计超大单仍为正或未明显回撤：允许洗盘
累计超大单下滑 + 主力/挂单风险同步出现：确认出场
硬止损只作为兜底
```

## 与 v0 的对照

v0 更偏“当天行为识别”。
v1 更偏“趋势结构识别”。

v1 的目标不是提高命中数量，而是提高：

- 中位收益
- 胜率
- 亏损票过滤能力
- 大趋势持有能力
