## 已否决 / 暂不采用链路

| 链路 | 结论 | 处理 |
|---|---|---|
| 旧确认/吸筹/出货三分打榜 | 业务解释弱，容易追急涨 | 归档 |
| 趋势中继入池后直接买 | 覆盖强但胜率低 | 只保留观察池，不直接交易 |
| 宽口径回踩承接 | 仍混入大量假中继 | 不投产 |
| 简单放宽成交额门槛 | 覆盖略增但质量下降 | 不改主策略 |
| Top20 扩容 | 覆盖增加但收益下降 | 不做默认 |
| 压单/OIB/CVD 单因子硬过滤 | 单因子不稳定或误杀大 | 只作为组合风险参考 |
| 大盘环境硬开关 | 过滤过重 | 暂不做开关 |
| 买入后 1~3 日早退 | 容易卖飞 | 不采用 |
