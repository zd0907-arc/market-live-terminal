## 策略人话解释

这套策略不是追当天暴涨，而是：

```text
先发现资金开始异动
再确认股价启动
然后等回调时有没有资金承接
确认承接后次日买入
买入后盯累计超大单有没有撤退
如果买入前多个风险信号叠加，就过滤掉
```

## 策略字段解释

| 字段 | 业务含义 |
|---|---|
| `setup_score` | 发现日资金/价格结构是否适合观察 |
| `launch3_return_pct` | 启动窗口涨幅，判断是否真的启动 |
| `pullback_support_spread_avg` | 回调时买盘承接是否强于卖压 |
| `confirm_distribution_score` | 确认日是否有出货风险 |
| `risk_count` | 多个风险信号同时出现的数量 |
| `risk_labels` | 具体风险原因 |
| `final_cum_super_amount` | 买入后累计超大单净流入 |
| `final_super_peak_drawdown_pct` | 累计超大单从峰值回撤比例 |

## 当前限制

- 完整挂单数据目前主要覆盖 2026-03 以后。
- 当前回测样本只有两个月，适合先做实盘观察/模拟盘，不应直接重仓依赖。
- 利通电子 3~4 月这类已经涨过一段再继续主升的票，更适合后续“趋势中继策略”。
