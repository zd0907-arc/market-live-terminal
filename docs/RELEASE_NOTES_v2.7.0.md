# v2.7.0 Release Notes

## 核心更新 (Key Features)

### 1. 散户情绪智能分析系统 (Retail Sentiment Intelligence)
本版本重磅推出了全新的“散户情绪”模块，将非结构化的股吧评论转化为可视化的量化指标。

*   **AI 舆情摘要**: 
    *   接入 LLM (大语言模型)，一键生成“A股游资风格”的市场情绪辣评。
    *   支持查看历史摘要记录，复盘不同时间点的市场观点。
*   **智能抓取引擎**:
    *   **14天深度回溯**: 首次抓取自动回溯过去两周的评论数据，建立完整趋势基准。
    *   **增量更新**: 智能识别已抓取数据，后续更新仅耗时毫秒级。
*   **多维趋势分析**:
    *   支持 **72小时 (小时级)** 与 **14天 (天级)** 两种时间维度的情绪热度与多空比趋势图。
*   **实战词库升级**:
    *   基于 1000+ 条真实评论挖掘，新增 "吃肉", "织布", "核按钮", "格局" 等 60+ 个实战黑话关键词，评分更精准。

### 2. UI/UX 深度优化
*   **交互式配置**: 
    *   全新的多标签页配置面板，支持在前端直接修改“情绪词库”和“AI API Key”，即时生效。
    *   修复了配置弹窗被浏览器遮挡的问题，改为跟随按钮的 Dropdown 模式。
*   **信息密度提升**:
    *   情绪模块采用紧凑的“左2右1”布局，大幅提升屏幕利用率。
    *   新增 Tooltip 解释浮窗，帮助用户理解“情绪得分”与“多空比”的算法逻辑。

### 3. API 接口变更
*   新增 `/api/sentiment/*` 系列接口，支持抓取、分析、摘要生成与历史查询。
*   `GET /api/sentiment/trend` 新增 `interval` 参数，支持 `14d` 聚合。

## 升级指南 (Upgrade Guide)
1.  **数据库迁移**: 启动后端时会自动创建 `sentiment_comments` 和 `sentiment_summaries` 表。
2.  **配置更新**: 建议在设置面板中填入 LLM API Key 以启用 AI 摘要功能。
