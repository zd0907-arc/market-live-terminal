根据项目结构和 `README.md`，以下是启动 **Market Live Terminal** 项目的计划。

该项目包含一个基于 Python FastAPI 的后端和一个基于 React/Vite 的前端。

### 1. 后端启动 (Backend Setup)
我们将设置并启动负责数据采集和持久化的后端服务。
1.  创建一个新终端 (Terminal 1)。
2.  安装 Python 依赖: `pip install -r backend/requirements.txt`
3.  启动 FastAPI 服务: `python -m backend.app.main`
    *   服务将运行在 `http://127.0.0.1:8000`。

### 2. 前端启动 (Frontend Setup)
我们将设置并启动前端开发服务器。
1.  创建第二个终端 (Terminal 2)。
2.  安装 Node.js 依赖: `npm install`
3.  启动 Vite 开发服务器: `npm run dev`
    *   前端将运行在 `http://localhost:3001`。

### 3. 验证 (Verification)
*   我们将检查后端进程输出以确保其正常运行。
*   我们将检查前端开发服务器输出，并提供可访问的链接。
