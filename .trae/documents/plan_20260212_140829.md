# UI 修复计划：散户情绪模块暗黑模式适配

## 1. 核心问题定位
用户反馈界面样式突兀，主要原因是**趋势图表组件 (`SentimentTrendChart`) 仍然使用了白色背景**，与应用的整体暗黑主题（Dark Mode）严重冲突。这导致左侧布局出现了一块显眼的“白色补丁”。

## 2. 修复方案 (Fix Plan)

### 2.1 趋势图组件重构 (`SentimentTrendChart.tsx`)
*   **移除背景**: 删除硬编码的 `bg-white`，使其背景透明。
*   **适配暗色坐标轴**: 将 X/Y 轴文字颜色 (`tick`, `label`) 修改为浅灰色 (`#94a3b8`)。
*   **适配暗色网格**: 将 `CartesianGrid` 线条颜色调暗 (`#334155`)。
*   **适配暗色 Tooltip**: 自定义 `Tooltip` 样式，使用深色背景 (`#1e293b`) 和浅色文字。

### 2.2 仪表盘容器调整 (`SentimentDashboard.tsx`)
*   **统一容器风格**: 将包裹图表的 `div` 背景从 `bg-white` 修改为 `bg-slate-800/50`，并添加统一的边框 (`border-slate-700/50`)，使其与上方的指标卡片风格一致。

## 3. 预期效果
*   左侧图表将完美融入暗黑背景，不再有白色色块。
*   坐标轴和图例清晰可见，不再刺眼。
*   整体视觉层次感更强，左右分栏更加协调。

## 4. 执行步骤
1.  修改 `SentimentTrendChart.tsx`：应用 Recharts 暗色主题配置。
2.  修改 `SentimentDashboard.tsx`：更新图表容器样式类名。
