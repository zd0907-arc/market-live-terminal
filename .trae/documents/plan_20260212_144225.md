# 情绪词库扩充计划 (Sentiment Keyword Mining)

## 1. 目标
*   抓取 5 只热门股票的至少 1000 条评论数据。
*   通过分析这些真实评论，提取新的高频“黑话”和情绪词。
*   更新系统配置，提升情绪评分准确度。

## 2. 选股策略 (Target Stocks)
选取以下 5 只具有代表性的高热度股票：
1.  **东方财富 (300059)**: 散户大本营，评论量大且情绪波动剧烈。
2.  **贵州茅台 (600519)**: 核心资产代表，多空分歧明显。
3.  **中信证券 (600030)**: 牛市旗手，情绪风向标。
4.  **宁德时代 (300750)**: 赛道股代表。
5.  **粤桂股份 (000833)**: 近期妖股代表（基于您的历史操作），容易出现极端情绪词。

## 3. 执行步骤

### Step 1: 批量抓取脚本 (`scripts/mine_keywords.py`)
*   编写一个独立脚本，不依赖 Web 服务。
*   功能：
    *   循环遍历上述 5 只股票。
    *   调用 `SentimentCrawler` 连续抓取（翻页抓取逻辑需要临时增强，或者多次触发首页抓取）。
    *   目标是每只股票抓取约 200 条，总量 > 1000 条。
    *   将数据存储到 `sentiment_comments` 表中。

### Step 2: 词频分析与提取
*   在脚本中增加简单的 NLP 分析功能（基于 `jieba` 分词或简单的频率统计）。
*   统计出现频率最高但未在现有词库中的词汇。
*   **人工/LLM 辅助筛选**: 
    *   将高频词和相关例句打印出来。
    *   使用 LLM (如果配置了) 或人工规则识别其情感倾向（多/空）。

### Step 3: 更新配置
*   生成新的推荐词库列表。
*   自动或手动更新 `app_config` 中的 `sentiment_bull_words` 和 `sentiment_bear_words`。

## 4. 预期产出
*   新增约 20-30 个实战情绪词（如：`T飞`, `做T`, `格局`, `瀑布`, `织布`, `骗线` 等）。
*   情绪评分算法的覆盖率和准确率显著提升。
