这是一个 React 前端 + Python 后端的混合项目，用于监控股票资金流向。

### 1. 核心架构
- **Frontend (UI)**: React + Vite (运行在端口 3000)
  - 负责界面展示、实时行情 (腾讯/东财接口)
  - 调用本地 Python 后端获取历史数据
- **Backend (Data)**: Python FastAPI (运行在端口 8001)
  - 负责作为数据代理，绕过 CORS 限制
  - 从新浪财经获取历史资金流向数据

### 2. 启动计划
我将按以下步骤为你启动项目：

#### 第一步：安装依赖
- **后端依赖**: 运行 `pip install -r requirements.txt` (安装 fastapi, uvicorn, requests 等)
- **前端依赖**: 运行 `npm install` (安装 react, vite, recharts 等)

#### 第二步：启动服务
我们将需要两个终端窗口分别运行前后端：
1.  **启动后端**: `python server.py`
    - 服务将启动在 `http://0.0.0.0:8001`
2.  **启动前端**: `npm run dev`
    - 服务将启动在 `http://localhost:3000`
